「无损放大」功能的注意事项

真的好想换手机啊啊啊啊……

「无损放大」这个功能是我和 波诺先生 从最开始就想做的一个功能，当时我还觉得这个功能如果不开个通向 Waifu2x 服务器的 API 的话是完全没法做的，不过好在现在有了 Core ML 这个东西可以让我很简单地在本机实现这个功能。

拖了这么久第二版的无损放大可算是弄完了（笑）

这次把所有设置都给堆上来了，上个版本放大效果不是很理想的原因是降噪开的太低了，以后如果要放大的话尽量直接选「Lunatic」级别的降噪，也就是最强降噪。

下面是对于运行时间的说明。

当看到你们的运行时间的时候我真是惊到了，X 不愧是新机型，比我的 6 强了不是一点半点。

中间是图片像素数，右边是运行时间

在这种情况下我觉得我的时间预测函数已经没有存在的意义了，于是我将其移除了。本来我是打算做个进度条之类的东西来着，后来发现这东西还挺难做的…… 于是这一版就暂时不加进度条了。

在看过你们的运行结果后我已经不知道要把上限设定成多少了，于是我决定不设定上限，崩不崩一听天命好了。
